Installation Guide
Getting Started with Aether
1
Download & Install
Run the installer above. Aether will install to your Applications folder and create a desktop shortcut.
2
Install Ollama
Aether uses Ollama for local AI processing. If it's not already installed, Aether will guide you through setup on first launch.
ollama.com/download
3
Activate Pro (Optional)
Open Aether, go to Settings, and paste your license key. Pro unlocks advanced sorting, vector memory, and project summaries.
4
Drop a Folder
Select any folder and let Aether read, classify, and organise your files. Your first scan takes a few minutes while the AI model initialises.
